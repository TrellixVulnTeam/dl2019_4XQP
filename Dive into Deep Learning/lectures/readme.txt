# Installation

## Install miniconda first 

conda create --name deepnumpy 
conda activate deepnumpy 
conda install pip jupyter rise

## Get mxnet build link from numpy.mxnet.io, e.g. MacOS  

pip install https://apache-mxnet.s3-accelerate.amazonaws.com/dist/python/numpy/latest/mxnet_mkl-1.5.0-cp37-cp37m-macosx_10_11_x86_64.whl  
pip install git+https://github.com/d2l-ai/d2l-en@numpy2
wget numpy.d2l.ai/d2l-en.zip
